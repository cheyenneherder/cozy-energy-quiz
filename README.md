# Cozy Energy Quiz

A simple 7-question personality quiz built with Python.

This project determines your current "energy phase" based on your answers, such as whether you are in a period of rest, focus, growth, or expansion.

## Features

* 7 reflective, mood-based questions
* Personalized result based on user responses
* Clean and simple command-line interaction

## What I Practiced

* Python input and output
* Lists for storing user responses
* Conditional logic (if/elif/else)
* Basic data handling using `.count()`

## How to Run

1. Make sure Python is installed
2. Run the script in your terminal:

```
python3 quiz.py
```
